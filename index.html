⚠️ HTTPS Required

Camera access requires a secure HTTPS connection or localhost. This won't work on plain HTTP. Deploy to Vercel, Netlify, GitHub Pages, or test on localhost.

Real face recognition flow:
1. Add students in the Students tab
2. Start camera, select a student from the dropdown, click "Enroll" to capture their face
3. Click "Detect & Mark Attendance" to recognise and auto-mark

Uses face-api.js with 128-D face embeddings stored in Firebase Firestore with localStorage offline backup.

Technical Stack
Language: HTML + CSS + Vanilla JavaScript
No framework, no build step — just open the .html file in a browser

Face recognition: face-api.js v0.22.2 (TensorFlow.js-based)
Runs entirely in the browser, runs on CPU

Database: Firebase Firestore (cloud) + localStorage (offline)
All face descriptors encrypted in localStorage, synced to Firestore when online

Mobile: Works on iOS 11+ (Safari) and Android Chrome
Requires HTTPS or localhost. Camera access needed.
